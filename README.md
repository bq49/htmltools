# htmltools
基于layaair H5引擎，开发的Html标签读取及解析封装。
使用LayaAir进行H5游戏开发的过程中，发现引擎自带的几个Html组件均无法满足游戏中的使用。

正好本人游戏中需要有带Html标签的跑马灯内容，于是对此进行了一个简单的封装。
